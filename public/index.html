API Endpoints for Capstone

Home Page

HTTP Method Path Description
GET / Home page

Users

HTTP Method Path Description
GET /users Retrieve all users
GET /users/:id Retrieve a user by ID
POST /users/register Register a new user
PATCH /users/:id Update a user by ID
DELETE /users/:id Delete a user by ID
POST /users/login Allows a user to login

Rooms

HTTP Method Path Description
GET /rooms Retrieve all rooms
GET /rooms/:id Retrieve a room by ID
POST /rooms/insertRoom Add a new room
PATCH /rooms/:id Update a room by ID
DELETE /rooms/:id Delete a room by ID
